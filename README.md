## 🚀 التثبيت على Render

### 1️⃣ إعداد المشروع محلياً

```bash
# استنساخ المشروع
git clone <your-repo-url>
cd mobo-global-chat

# تثبيت التبعيات
npm install

# اختبار محلي
npm start
```

### 2️⃣ إنشاء حساب على Render

1. اذهب إلى [render.com](https://render.com)
2. سجل حساب جديد (مجاني)
3. اربط حسابك مع GitHub

### 3️⃣ رفع المشروع على GitHub

```bash
# إنشاء مستودع جديد
git init
git add .
git commit -m "Initial commit - MOBO Chat v5.0"
git branch -M main
git remote add origin <your-github-repo-url>
git push -u origin main
```

### 4️⃣ نشر على Render

1. في لوحة تحكم Render، اضغط **"New +"**
2. اختر **"Web Service"**
3. اربط مستودع GitHub الخاص بك
4. املأ البيانات:
   - **Name**: `mobo-chat` (أو أي اسم تريده)
   - **Environment**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Plan**: اختر **Free**

5. اضغط **"Create Web Service"**

### 5️⃣ الإعدادات الإضافية

في صفحة الخدمة على Render:
- انتقل إلى **Environment**
- أضف متغير البيئة (اختياري):
  ```
  PORT = 3000
  NODE_ENV = production
  ```

## 🎮 كيفية الاستخدام

### للمستخدمين العاديين:

1. **إنشاء حساب جديد**
   - اختر اسم مستخدم (3-20 حرف)
   - أدخل كلمة مرور قوية (4+ أحرف)
   - اختر الجنس (ذكر/أنثى)
   - اضغط "إنشاء حساب جديد"

2. **تسجيل الدخول**
   - أدخل اسم المستخدم
   - أدخل كلمة المرور
   - اضغط "دخول إلى الدردشة"

3. **الدردشة**
   - ستنضم تلقائياً للغرفة العالمية
   - اكتب رسالتك واضغط إرسال
   - يمكنك رؤية المستخدمين المتصلين

### للمديرين:

1. **كتم مستخدم**
   - اضغط على أيقونة 🔇 بجانب اسم المستخدم
   - حدد المدة بالدقائق
   - أدخل سبب الكتم

2. **حظر مستخدم**
   - اضغط على أيقونة 🚫
   - أكد الحظر
   - أدخل سبب الحظر

3. **حذف رسالة**
   - اضغط على أيقونة 🗑️ في الرسالة
   - أكد الحذف

## 🐛 استكشاف الأخطاء

### المشكلة: لا يعمل الموقع على Render

**الحل**:
1. تحقق من أن `package.json` موجود
2. تأكد من وجود `"start": "node server.js"` في scripts
3. تحقق من سجلات Render (Logs)

### المشكلة: خطأ في الاتصال بـ Socket.io

**الحل**:
1. تأكد من أن المنفذ صحيح
2. تحقق من إعدادات CORS
3. جرب WebSocket و Polling معاً

### المشكلة: الرسائل لا تظهر

**الحل**:
1. افتح Console في المتصفح
2. تحقق من الأخطاء
3. أعد تحميل الصفحة
4. تأكد من تسجيل الدخول بنجاح

## 🎓 معلومات تقنية متقدمة

### Architecture:
```
Client (Browser)
    ↓ WebSocket/HTTP
Server (Node.js + Express)
    ↓ Socket.io
Real-time Communication Layer
    ↓
In-Memory Data Store (Map Objects)
```

### Event Flow:
```
1. User connects → Socket established
2. Login/Register → Authentication
3. Join room → Room assignment
4. Send message → Broadcast to room
5. Disconnect → Cleanup resources
```

### Security Layers:
```
Layer 1: Input Validation
Layer 2: HTML Sanitization
Layer 3: Password Hashing (bcrypt)
Layer 4: Session Management
Layer 5: Rate Limiting
```
